Docker Tutorial

Installing Docker on Windows

  1. Download the Docker Desktop Installer here
  2. Make sure you have:
    • Windows 10 or later
    • Enabled Hyper-V and Container Windows features (instruction)
    • Up to Date Linux kernel WSL2 Package (download)

    Setting up Apache Image in Windows

    1. Open up the Command Prompt, type and run the following command “docker pull ubuntu”.
    2. Type and run “docker run -it ubuntu:latest /bin/bash” command to make a container and run the image.
    3. Type the command “apt-get update” to install the Ubuntu updates.
    4. Type the command “apt-get install apache2” to download Apache web server. Make sure to:
      • Type ‘Y’ when asked, to move on.
      • Choose your geographical location which is the region your machine currently resides in.
      • Select the time-zone of your machine.
    5. Then to install your document editor, type in “apt-get install vim”.
    6. After all is set up, open another command prompt instance.
    7. Type “docker ps” in the new command prompt.
    8. Type “docker run -it -p 127.0.0.1:80:80 -v FolderOnMachine:/var/www/html new image name /bin/bash” to boot up your Apache Web Server.
      • Preferably set up the folder in Windows (C:), don’t do it in Downloads, Desktop, Images, etc.
    9. Type and run “cd /var/www/html”
    10. Type “vim index.html” to make a index.html file.
    11. Press 'i'
    12. Type in basic html site content.
    13. Return to the command prompt then type “:wq” finally press Enter.

      14. Eevrything is setup!

    Viewing Your Local Website

    To view your local server, open your browser and type in the IP in step 10. For example, it will be ‘127.0.0.1’ (excluding :80:80). You should be able to see your site now.
